Privacy Policy

Last updated: January 21, 2026

This application ("the App") provides document translation and multilingual content generation using artificial intelligence. This Privacy Policy explains how the App handles user data and protects your privacy.

1. Information We Collect

The App does not collect, store, or retain any personal information about users.

The App may temporarily process the following data solely for the purpose of providing its functionality:

  • Text extracted from user-provided documents or snapshots
  • User-entered text for translation
  • Generated multilingual output
  • PDF files created by the user

This data is processed only to perform translations and generate results. No data is stored on external servers, databases, or cloud services.

2. How Your Data Is Used

User-provided content is used exclusively to:

  • Translate text into selected languages
  • Generate multilingual information
  • Create PDF files upon user request

All processing is performed automatically. The App does not analyze, track, or profile users.

3. Data Storage and Retention

The App does not store any user data permanently.

  • No documents, translations, or PDFs are saved by the developer
  • No data is uploaded to external servers
  • No backups or logs of user content are kept

Any data processed by the App exists only temporarily during use and is deleted automatically when the App is closed or the process is completed.

4. Data Sharing

The App does not share any user data with:

  • Third parties
  • Advertisers
  • Analytics providers
  • External services

Users may choose to share exported PDF files themselves (e.g., via email or messaging apps). This sharing is entirely user-initiated and outside the control of the developer.

5. Children's Privacy

The App is not intended for children under 13 and does not knowingly collect personal information from children.

If you believe a child has submitted personal information through the App, please contact us so we can remove it.

6. Security

The App uses industry-standard methods to ensure that all processing is handled securely on the device. Since no data is stored or transmitted, the risk of unauthorized access is minimized.

7. Changes to This Policy

This Privacy Policy may be updated from time to time. Any changes will be reflected in the updated version of this document.
